NAOMI Notes

NAOMI's changes

NAOMI's general purpose is to change the way multiple weapons work to resemble that of the original Japanese release of DXMP.

Below is a list of all the changes NAOMI makes. because NAOMI is still in development, this list is subject to change at any time.

  • Modified damage to Rifle, Pistol and Melee Low-Tech skill weapons.

  • Halved bioelectricity drain.

  • Updated weapon projectiles. (visual, performance)

  • Updated the way Heavy skilled weapons work.

  • Modified the way the "targeting" augmentation works.

  • Updated the games scoreboard, skill buying menu, and killerprofile / death screen.

Compatibility

NAOMI can be tweaked to suit a number of playstyles with the bundled NAOMI.ini configuration file, these values are:

  • FrobDistance* This defines the FrobDistance (select & use distance) of a player.
    It is advised to use the default value.
    Default parameter: 112.00
    Accepted parameters: Any number
  • bEnableSpectating* Allows or disallows player spectating while in game.
    Default parameter: True
    Accepted parameters: True or False
  • bResetSkin* Forces correct skin of a player during team games, it is recommended to keep this enabled.
    Default parameter: True
    Accepted parameters: True or False
  • bSlowAugDrain Uses the Japanese original bioelectricity multiplier, for more accurate feel.
    Default parameter: True
    Accepted parameters: True or False
  • bNoAugDrain Disables the draining of bioelectricity in multiplayer, for a more "arcade-y" feel.
    Default parameter: False
    Accepted parameters: True or False
  • bGodHandMode Replaces the pistol with the godhand. get some.
    Default parameter: False
    Accepted parameters: True or False
